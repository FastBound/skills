# Contributing

Thanks for your interest. These Agent Skills are maintained and published by
FastBound, Inc. as proprietary, first-party material (see [LICENSE](./LICENSE)).

**We do not accept external pull requests, forks-for-redistribution, or
derivative works.** The license permits you to download, install, use, and
modify the skills locally for your own FastBound integrations, but not to
redistribute them. Please don't open a PR with changes — it can't be merged.

## Found a bug, an error, or something out of date?

We genuinely want to hear about it. The skills are kept accurate against the
FastBound API, and corrections are welcome as **reports**, not code changes:

- **Skill bugs, inaccuracies, or suggestions:** email support@fastbound.com, or
  open a GitHub issue describing the problem (which skill, which file, what's
  wrong, and what you expected).
- **Security issues:** do not open a public issue — see [SECURITY.md](./SECURITY.md).
- **Partnership, redistribution, or embedding rights:** email
  support@fastbound.com to start that conversation.

Each skill's own `SKILL.md` and reference files are the place for usage details
(authentication, endpoints, credentials, and so on) — this file is only about
how to engage with the repository.
